PRD – Fruit Basket Selection Software

1. Product Name:
Fruit Basket Selector

2. Purpose:
A software/app that allows users to select fruits and add them to a virtual basket before buying.

3. Target Users:

Online grocery customers

Fruit shop owners

Delivery platforms

4. Problem Statement:
Customers want an easy way to choose fruits online and manage their basket before checkout.

5. Solution:
A simple application where users can view fruits, select quantity, and add them to a basket.

6. Key Features:

Display list of fruits

Add fruit to basket

Remove fruit from basket

Change quantity

View total price

7. Functional Requirements:

User can see available fruits.

User can select fruit and quantity.

System adds fruit to basket.

Basket shows all selected fruits and total cost.

User can remove or update items.

8. Non-Functional Requirements:

Fast response time

Simple user interface

Secure payment integration

Works on mobile and web

9. Success Criteria:

Easy fruit selection

Quick checkout

Reduced user effort

10. Example Workflow:

User opens app

Selects fruits (Apple, Banana, Mango)

Adds them to basket

Views basket

Proceeds to checkout 🍎
